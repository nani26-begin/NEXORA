## ✨ Overview

**NEXORA** is a sleek, production-ready web application crafted with the latest cutting-edge frontend technologies. It combines the power of Next.js 16 App Router with React 19's concurrent features, smooth animations via Framer Motion and GSAP, all styled with the new Tailwind CSS v4.

## 🛠️ Getting Started

### Prerequisites

Make sure you have the following installed:

- **Node.js** >= 18.x
- **npm** >= 9.x (or yarn / pnpm)

### Installation

1. **Clone the repository**

```bash
git clone https://github.com/nani26-begin/NEXORA.git
cd NEXORA
```

2. **Install dependencies**

```bash
npm install
```

3. **Start the development server**

```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

```bash

```
